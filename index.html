Health_Monitering_system

The Health Monitoring System is a scalable platform designed for diagnostic centers to analyze critical health metrics like blood pressure, sugar level, cholesterol, and hemoglobin. It processes synthetic data for 10,000+ patients using Apache Spark and Pandas, identifying trends and high-risk individuals through statistical analysis.

The system includes data visualization tools to uncover patterns and correlations in health data, making it suitable for real-world integration. Its modular design and big data capabilities make it a strong candidate for future enhancements like ML-based disease prediction and real-time monitoring.

Heart Disease Prediction

Heart Disease Prediction is a machine learning-powered web app that forecasts the likelihood of heart disease based on user inputs like age, chest pain type, and maximum heart rate. It combines a logistic regression model with a simple UI to deliver fast, reliable results.

The app features a responsive design, real-time predictions, and error handling, making it accessible and user-friendly. Built with Flask, HTML/CSS, and Scikit-learn, it's ideal for educational purposes or early-stage health diagnostics through web technology.

COVID-19-Data-Dashboard-and-Analysis

COVID-19 Dashboard is a comprehensive data visualization project built with Tableau that provides real-time insights into global and regional COVID-19 trends. It displays key metrics like confirmed cases, recoveries, deaths, and vaccination progress with interactive filters for date, geography, and demographics.

The dashboard uses live data sources such as the Johns Hopkins University dataset and WHO reports. Designed to be both informative and user-friendly, it supports drill-down capabilities and trend analysis to better understand the pandemic's impact worldwide.
